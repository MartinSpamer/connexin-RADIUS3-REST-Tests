/accounting_entries

Methods

GET

Returns a list of Accounting Entries

request query parameters
parameter value description

page

integer

Default: ${#TestCase#pageNo}

Page number of results to be retrieved.

limit

integer

Default: ${#TestCase#limit}

The maximum number of records to be return per page

user

integer

Default: ${#TestCase#userId}

The unique ID for the User who's Accounting Entries are retrieved

username

string

Default: ${#TestCase#username}

The RADIUS Username for Accounting Entries (depreciated)

page

integer

One of:

  • ${#TestCase#PageNo}
  • 1

The page number of results to be retrieved (default: 1)

limit

integer

One of:

  • ${#TestCase#PageLimit}
  • 25

limit - The maximum number of records to retrieved per page (default: 25)

available response representations:

  • Status Code200-application/json
  • Status Code200-application/json; charset=UTF-8
  • Status Code400-text/html; charset=UTF-8
